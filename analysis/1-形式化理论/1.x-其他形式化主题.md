### 1.x.4 形式化定义与证明

#### HoTT中的等价类型（Lean 代码示例）

```lean
structure Equiv (A B : Type) :=
  (to_fun : A → B)
  (inv_fun : B → A)
  (left_inv : ∀ x, inv_fun (to_fun x) = x)
  (right_inv : ∀ y, to_fun (inv_fun y) = y)
```

#### 量子比特的LaTeX表达

$$
|\psi\rangle = \alpha|0\rangle + \beta|1\rangle,\ \alpha,\beta \in \mathbb{C},\ |\alpha|^2 + |\beta|^2 = 1
