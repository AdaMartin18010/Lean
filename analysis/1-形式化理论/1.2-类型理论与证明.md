### 1.2.5 典型案例与实现

#### 自然数类型与归纳证明（Lean 代码示例）

```lean
inductive Nat
| zero : Nat
| succ : Nat → Nat

theorem add_zero (n : Nat) : n + 0 = n :=
begin
  induction n,
  case zero { refl },
  case succ k ih { simp [ih] }
end
```
