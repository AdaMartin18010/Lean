### 1.2.6.4 形式化定义与Lean证明

```lean
-- 自然数归纳证明示例
theorem add_zero (n : Nat) : n + 0 = n :=
begin
  induction n,
  case zero { refl },
  case succ k ih { simp [ih] }
end
```

#### 归纳法证明（LaTeX表达）
