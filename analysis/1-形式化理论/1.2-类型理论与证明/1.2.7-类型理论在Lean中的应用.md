### 1.2.7.4 代码实现与自动化

```lean
-- 归纳定义自然数与加法
def add : Nat → Nat → Nat
| Nat.zero, n => n
| Nat.succ m, n => Nat.succ (add m n)

-- 自动化证明加法交换律
theorem add_comm (m n : Nat) : add m n = add n m :=
begin
  induction m,
  case zero { simp [add] },
  case succ k ih { simp [add, ih] }
end
```
